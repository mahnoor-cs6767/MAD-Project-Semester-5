## 🧠 Project Description

This project demonstrates the conversion of the **Little Learners educational website** into a **mobile application** using **React Native** and **Snack Expo**.  
The app provides kids with an interactive platform to **learn, play, and explore** through cards, games, and quizzes.  
It also includes **user authentication and account management** using **Supabase backend connectivity**.

The project includes:
- **Splash Screen** with animations and user session check  
- **Login & Create Account screens** with Supabase authentication  
- **Main navigation** for cards, games, and quizzes  
- **Database connectivity** using Supabase for storing and verifying user accounts  
- **Multiple screens** for Stories, Hadiths, Duas, Words, Games, and Quizzes  

---

## 🛠️ Technologies & Tools Used

- **Framework:** React Native (Expo)  
- **Backend:** Supabase (Authentication & Database)  
- **Libraries:** React Navigation, AsyncStorage, React Native Paper, Linear Gradient  
- **Development Platform:** Snack Expo  
- **Assets:** Images, icons, and custom styling
